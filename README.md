# nim examples

* Requirements
  * nim >= 1.0.0
  * nimble >= v0.11.0

# Run code

## fizzbuzz

```bash
cd fizzbuzz
nim compile --run fizzbuzz.nim
# or
nim c -r fizzbuzz.nim
```

## greetings

```bash
cd greetings
nim c -r greetings.nim
```

## nim_hello

```bash
cd nim_hello
nimble install -y
nimble run nim_hello
```

## http_client

```bash
cd http_client
nimble -d:ssl run http_client
```
